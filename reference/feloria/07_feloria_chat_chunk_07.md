1. 타일이 반복적으로 이상하게 깔리는 문제

플레이어: 따로 렌더
즉, 맵 JSON 구조를 쓰면 이 문제를 고칠 수 있습니다.

Use the provided JSON map layouts together with the tileset atlas system.

Do not render all map content as a single flat layer.

1. groundLayer
2. detailLayer
3. objectLayer with depth sorting when needed
4. player sprite
5. foreground objects
6. UI overlays such as dialogue box and area label separately

Make sure the dialogue box is not blended into the world layer.
It must be rendered as a separate screen UI element.
