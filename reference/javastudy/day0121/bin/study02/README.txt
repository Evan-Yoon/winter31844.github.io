<Data Labeling Application>

JavaFX를 사용하여 텍스트 데이터를 읽어오고, 감성 분석을 위한 학습 데이터를 구축하는 프로그램입니다.
대화 내용을 문장 단위로 확인하고 긍정, 중립, 부정으로 분류하여 파일로 저장하는 기능을 수행합니다.


[주요 기능]

-데이터 로딩 및 파싱: data.txt 파일에서 대화 내용을 자동으로 불러와 라벨링 하기 좋은 형태로 정제합니다.
-3단계 라벨링:
   - 긍정 : 라벨값 1 저장
   - 중립 : 라벨값 2 저장 (직접 기능 추가)
   - 부정 : 라벨값 0 저장
-자동 저장: 버튼을 누르는 즉시 labeled_data.txt 파일에 결과가 기록됩니다.
-진행 상황 표시: 전체 데이터 중 현재 작업 위치를 실시간으로 보여줍니다.


[UI 디자인 (with Gemini)]

전반적인 UI 구성과 CSS 스타일링은 생성형 AI(Gemini)를 활용하여 작업했습니다.
-가독성 개선: 긴 문장도 한 줄에 잘 보이도록 창 크기를 조정하고, 텍스트가 눈에 잘 띄도록 카드 형태의 레이아웃(Card UI)을 적용했습니다.
-색상 적용: 장시간 작업에도 눈이 편안하도록 배경색을 조정하고, 각 버튼의 성격(긍정-파랑, 중립-초록, 부정-빨강)에 맞춰 직관적인 그라데이션 컬러를 입혔습니다.

[기능 구현 : 중립(Neutral) 옵션 추가]

초기 기획에는 긍정과 부정 분류만 있었으나, 실제 데이터를 확인해 보니 어느 쪽에도 속하지 않는 애매한 문장이 많아 '중립' 분류가 필수적이라고 판단했습니다.
이에 따라 직접 코드를 수정하여 기능을 확장했습니다.
-UI 수정: 긍정과 부정 버튼 사이에 '중립' 버튼을 배치했습니다.
-로직 구현: rootController.java에 핸들러(onActionNeutralBtn)를 추가하고, 파일 저장 시 라벨값 2로 구분되어 저장되도록 FileManager 연동 로직을 작성했습니다.

[프로젝트 구조]

src/study02
-AppMain.java : 메인 실행 파일
-root.fxml : UI 레이아웃 (View)
-rootController.java : 버튼 이벤트 및 로직 처리 (Controller)
-application.css : 디자인 스타일시트
-FileManager.java : 파일 입출력 처리 클래스
-data.txt : 원본 데이터
-labeled_data.txt : 라벨링 결과 파일

[실행 방법]

1.JDK 11 이상 및 JavaFX 환경이 필요합니다.
2.IDE에서 프로젝트를 엽니다.
3.AppMain.java를 실행합니다.
4.화면에 뜨는 문장을 읽고 적절한 감정 버튼을 클릭하여 분류 작업을 진행합니다.
