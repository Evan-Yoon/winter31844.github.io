# 3차 프롬프트

이제 마지막 배경 자료를 넣을게.
이번에는 **상태 관리 / 저장 / 기능 확장 / 마감 폴리싱 / 느낀 점**까지 연결해서, 최종 블로그의 뼈대를 잡아줘.

## 작업 목표
- 근무 입력 → 계산 → 합계 반영 흐름을 설명하기
- 이 앱이 단순 계산기에서 점점 제품처럼 확장된 흐름을 정리하기
- 커밋 히스토리를 블로그 서사에 자연스럽게 녹일 방법 찾기
- 느낀 점 / 배운 점 / 다음 개선점까지 연결하기

## 출력 형식
1. 전체 블로그 아웃라인 1개
2. 각 섹션별 핵심 메시지 1~2줄씩
3. 커밋 히스토리를 블로그에 넣는 방식 제안 3가지
4. 느낀 점 / 배운 점 / 다음 개선점 문단 초안 각 1개씩
