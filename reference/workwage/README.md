# WorkWage blog input package

이 패키지는 Antigravity의 Claude에게 **한 번에 넣기 어려운 개발 기록**을 순차적으로 먹이기 위한 용도입니다.

## 구성
- `manifest.csv`: 전체 파일 목록과 역할
- `commit_map.csv`: 공개 커밋 히스토리를 블로그 서사에 맞게 매핑한 표
- `chunks/`: Claude에게 순서대로 넣을 서사형 요약 청크
- `prompts/`: 1차~4차 입력용 복붙 프롬프트
- `source_notes.md`: 어떤 자료를 근거로 만들었는지 설명

## 권장 투입 순서
1. 1차 프롬프트 + chunk 01~02
2. 2차 프롬프트 + chunk 03~05
3. 3차 프롬프트 + chunk 06~09 + commit_map.csv
4. 4차 프롬프트로 최종 블로그 초안 생성

## 핵심 의도
이 자료는 “기능 나열”보다 아래 흐름이 잘 보이도록 설계했습니다.
- 왜 이 앱을 만들었는가
- 어떻게 범위를 줄였는가
- 어떤 기술/로직이 핵심이었는가
- 기능이 어떻게 확장되었는가
- 무엇을 느끼고 배웠는가
