주제: GitHub Pages 기반 Astro 블로그에 Claude 자동 글 생성 붙이기

핵심 내용:
- 기존에는 글을 직접 md 파일로 작성했다
- 이제는 input.txt에 간단한 메모만 적고 자동으로 posts에 글이 생성되게 만들고 싶다
- Claude API 사용 예정
- style.md로 기존 말투 유지
- scripts/generatePost.js로 자동 생성
- src/content/posts 에 md 파일 저장
- GitHub Actions로 push 시 자동 실행

추가로 넣고 싶은 포인트:
- 왜 자동화를 하고 싶었는지
- 직접 글을 쓰는 것도 좋지만 꾸준히 기록하는 게 더 중요하다고 느낀 점
- 나중에는 프로젝트 회고나 트러블슈팅도 더 빠르게 쌓고 싶음
